# Logistic Cost Verification Project

## Project Overview
This project focuses on verifying and analyzing logistics costs using data analysis techniques. The dataset was cleaned and processed using Python and Excel to ensure data accuracy and consistency. Exploratory Data Analysis (EDA) was performed to identify patterns in transportation costs, delivery distances, and operational expenses. Visualizations were created to highlight cost variations and uncover insights that can help businesses optimize logistics operations and make more data-driven decisions.

## Tools Used
Python  
Pandas  
Microsoft Excel  
Power BI  
Matplotlib  

## Key Insights
• Identified patterns in transportation cost variations.  
• Analyzed the relationship between delivery distance and logistics cost.  
• Highlighted potential areas where logistics costs can be optimized.

## Project Goal
The goal of this project is to analyze logistics cost data and generate insights that help businesses improve operational efficiency, reduce unnecessary expenses, and support better decision-making in supply chain management.
